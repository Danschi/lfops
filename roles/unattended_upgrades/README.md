# Ansible Role linuxfabrik.lfops.unattended_upgrades

This role simply removes the `unattended-upgrades` package from the system.

Runs on

* Debian 11
* Ubuntu 16


## Tags

| Tag                   | What it does                              |
| ---                   | ------------                              |
| `unattended_upgrades` | Removes the `unattended-upgrades` package |
